Push Notification Client APP

This is a very simple example of an APP/Site that allows the client to register for push notifications.


Step 1: Create VAPID Keys

Run node create.js in the vapid-keygen folder to create a public/private key pair.

Step 2: Enter public key in the field below

Step 3: Push the button below!

Step 4: Use subscription JSON to push message

Run node push.js in the send-push folder to send push message. Enter the JSON below when asked
